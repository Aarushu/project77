Project On Console

Visual Studio Code (Vscode) is an IDE software that helps us to make webpages. To set it up, go onto the official vscode website and install it.

The console screen is an important screen. It is used for testing the code. To access it, Right click on your mouse and click on inspect. on The top-right corner, you will find the console. To print anything on the console, You need to use the js tag console.log().

To see the console of this screen, follow the steps above.
